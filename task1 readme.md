# Hangman Game

## Problem Statement:
Develop a simple text-based Hangman Game using Python where the player guesses a hidden word one letter at a time. The game should randomly select a word from a predefined list and allow the player a maximum of 6 incorrect guesses. The program should display the correctly guessed letters and update the word progress after each attempt. The game ends when the player guesses the complete word or uses all incorrect attempts.

## Features
- Random word selection
- 6 incorrect guesses allowed
- Console-based gameplay
  
## Tools Used
- Python
- Random Module

## Conclusion
This project helped in understanding Python concepts like loops, conditions, lists, strings, and random module usage through an interactive game.
