# Stock Portfolio Tracker

##Project Description
This is a simple Python project that helps users track their stock investments.  
Users can enter stock names and quantities, and the program calculates the total investment value using predefined stock prices.

##Features
- Add stock names and quantities
- Calculate total investment value
- Display portfolio summary
- Save results to a text file
- Uses Python dictionaries and file handling

## Technologies Used
- Python

##Conclusion
This project demonstrates a basic stock portfolio management system using Python. It helps beginners understand important programming concepts through a real-world example.
