##Project Description  
             This project is a simple Python automation script that moves all .jpg image files from a source folder to another folder automatically.
It uses Python's built-in os and shutil modules for file handling and file movement. The script also creates the required folders if they do not already exist. This project helps beginners understand basic task automation, file management, and folder operations using Python.

##Features
Automatically detects .jpg files
Moves image files from source folder to destination folder
Creates folders automatically if they do not exist
Simple and beginner-friendly Python project
Uses Python built-in modules (os and shutil)
Helps automate repetitive file management tasks

##Technologies Used
Python
os Module
shutil Module
File Handling
